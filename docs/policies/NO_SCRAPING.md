\# No Scraping Policy



Neptune/Pluto/Atlas must not scrape websites.



Allowed internet interactions:

\- Official APIs (authenticated, permissioned)

\- RSS/Atom feeds where offered

\- Webhooks and event subscriptions

\- User-directed browsing (open page; user clicks)

\- User-provided text (copy/paste) for analysis



Disallowed:

\- Automated extraction from arbitrary web pages (HTML parsing for data collection)

\- Automated scrolling/clicking to harvest content

\- Bypassing paywalls, login walls, or technical restrictions

\- Bot-like behavior intended to mimic humans for collection



All network features must be designed to comply with site terms and applicable law.
